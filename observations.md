# Observations

Looking at the top 10 recommendations generated for User 1, it's pretty clear that the content-based filtering model picked up on a strong preference for action-heavy, comedic adventure films, since every single result falls within the Action|Adventure|Comedy cluster — movies like *Kingsman: The Golden Circle*, *Machete*, and both *Teenage Mutant Ninja Turtles* entries all share that same broad genre overlap, which suggests the user's 232 ratings were skewed heavily toward that combination and the weighted profile vector ended up pulling almost exclusively in that direction. The similarity scores are also quite tightly packed between 0.80 and 0.84, which could mean the TF-IDF tag features aren't adding much discriminative signal for this particular user (possibly because the movies they watched don't have many tags, or the tags are too generic to push any single candidate significantly ahead of the others), so the genre one-hot features are doing most of the heavy lifting. One thing worth investigating in a follow-up would be whether normalising the genre weights or increasing the TF-IDF feature count would introduce more diversity into the recommendation list, as right now the system essentially recommends the same "type" of movie ten times over rather than surfacing anything surprising or serendipitous.
